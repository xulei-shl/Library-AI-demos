src\core\book_vectorization\excel_exporter.py，执行图书检索以及调用大模型主题筛选后。对通过的候选书目列表撰写总的推荐导语

- 大模型调用的逻辑请直接复用 src\utils\llm。大模型配置见 config\llm.yaml
- 系统提示词为 prompts\推荐导语.md（默认也是使用langfuse同名的提示词）
- 用户提示词包括2个部分：
    - 文章分析报告，是一个md文档，完整提取作为用户提示词的组成部分。md文档可以在命令行窗口中人工输入
    - 图书元数据：从excel中提取每一行的书目元数据，作为用户提示词的组成部分。excel文件也是命令行窗口中人工输入
        - 书目条码id（书目条码列）
        - 书名（豆瓣书名列）
        - 副标题（豆瓣副标题列）
        - 作者（豆瓣作者列）
        - 丛书（豆瓣丛书列）
        - 内容简介（豆瓣内容简介）
        - 作者简介（豆瓣作者简介）
        - 目录（豆瓣目录）
        - 初评理由（初评理由列）
- 大模型返回的结果保存到excel同路径的同名的md文档中

- 交互方式，在` python scripts/retrieve_books.py --interactive`中新增命令6 执行这个大模型推荐导语模块，用户需要输入文章分析报告的路径，以及excel文件路径。
