这个表中有哪些数据需要向量化，下面的条件必须同时满足（条件也是可配置的）：
- douban_summary、douban_title、douban_author、douban_catalog必须有值
- douban_rating根据call_no的首字母高于配置的分数：（这个规则也是可配置的）
```
        B: 8.2
        C: 7.5
        D: 7.1
        E: 7.4
        F: 7.1
        G: 7.3
        H: 7.2
        I: 7.3
        J: 7.7
        K: 8.0
        N: 7.6
        S: 7.4
        T: 7.6             
        default: 7.5 # 其他未列出分类的默认及格分
```
这2类条件同时满足才进行向量化存储
