# 书海回响图书推荐系统 - 学术论文实验系统分析执行计划

## 分析目标
将图书馆业务逻辑与AI技术融合的过程转化为学术语言，详细记录从"原始数据获取"到"辅助决策产出"的全链路，为论文提供详实的实验支撑。

## 项目模块识别（基于程序运行输出）
根据main.py程序运行输出，系统包含以下核心模块：

### 活跃模块分析范围：
1. **模块1/2-借阅模块**: 月归还数据分析 + 智能筛选
2. **模块6-新书模块**: 零借阅（睡美人）筛选  
3. **模块3-B-公共模块**: 豆瓣模块（FOLIO ISBN + 豆瓣 ISBN API + 配置是否启用动态评分过滤）
4. **模块6-B-新书模块**: 新书评分过滤（写入候选状态）
5. **模块4-公共模块**: 初评（海选阶段）
6. **模块4-公共模块**: 完整评选（初评→决选→终评）
7. **模块5-公共模块**: 图书卡片生成（含借书卡）
8. **模块7-主题模块**: 主题书目每日追踪

### 排除模块：
- **模块3-废弃保留**: 豆瓣模块（已明确废弃，不纳入分析）
