改动汇总
1. 配置文件 config/setting.yaml
新增 rating_filter 配置块（第500-544行）：
rating_filter:
  dynamic_filter_enabled: true  # 借阅数据设为true，新书设为false
  
  new_sleeping:  # 新书专用过滤规则
    category_min_scores: {...}  # 不同学科最低评分
    required_fields: [...]       # 必填字段验证
    pub_year_filter: {...}       # 出版年过滤
    reuse_filters: {...}         # 复用现有txt过滤规则

2. 流水线 douban_isbn_api_pipeline.py
修改 _run_rating_filter 方法，优先读取配置开关：
如果 rating_filter.dynamic_filter_enabled=false，跳过动态过滤

3. 新建 src/core/new_sleeping/rating_filter.py
新书专用评分过滤器，包含5个规则：
必填字段验证（封面、简介）
出版年过滤（当年）
题名关键词过滤（复用 title_keywords.txt）
索书号规则过滤（复用 call_number_clc.txt）
学科最低评分过滤

4. 主入口 main.py
新增 run_module6b() 函数
菜单新增 10. 模块6-B: 新书评分过滤（写入候选状态）
退出选项改为 12

场景	配置修改	命令流程
