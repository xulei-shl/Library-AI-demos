## HTML模板截图兼容性规则 ⚡ **严格遵循**

### 🎯 **主容器CSS类名规范**
```html
<!-- 选择器优先级（从高到低） -->
<div class="poster">内容</div>           <!-- 最高优先级，适用于需要额外底部空间的模板 -->
<div class="library-card">内容</div>     <!-- 借书卡模板，使用精确边界 -->
<div class="book-card">内容</div>        <!-- 图书卡片模板，使用精确边界 -->
<div class="card">内容</div>             <!-- 通用卡片模板，使用精确边界 -->
```

### 📏 **布局结构要求**
- **主容器必须是body下的第一个div元素**
- **容器尺寸限制**：宽度300-1000px，高度200-2000px
- **DOM结构**：`<body><div class="xxx">内容</div></body>`

### 🖼️ **图片处理规范**
- **所有图片必须放在`pic/`目录下**
- **使用相对路径**：`<img src="pic/cover.jpg">`
- **支持格式**：PNG, JPG, JPEG, GIF, WebP
- **文件大小**：建议5MB以内

### 🎨 **视觉效果注意事项**
- **`.poster`模板**：自动增加底部50px，防止内容被截断
- **其他模板**：使用精确边界，不会有额外空白
- **避免过大的阴影效果**（可能被裁切）
- **重要内容距离容器底部至少60px**

### ✅ **标准模板结构示例**
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>图书卡片</title>
    <style>
        /* CSS样式定义 */
    </style>
</head>
<body class="bg-gray-100 flex items-center justify-center min-h-screen p-4">
    <!-- 主容器：必须使用优先级最高的类名 -->
    <div class="poster">
        <!-- 所有内容都在这里 -->
        <div class="content">
            <h1>{{TITLE}}</h1>
            <p>{{AUTHOR}}</p>
            <p>{{RECOMMENDATION}}</p>
            <img src="pic/cover.jpg" alt="封面">
        </div>
    </div>
</body>
</html>
```
