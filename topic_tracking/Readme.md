# 学术定题追踪与 AI 问答

## 主要功能包括:

1. 主题词管理:
    - 用户可以管理感兴趣的论文主题关键词
    - 脚本会根据关键词从 CNKI 爬取核心期刊相关论文元数据

2. 文献库管理:
    - 对爬取的论文元数据(标题、作者、摘要等)进行存储和管理
    - 用户可以查看、检索和导出文献信息（其实都是本地文件存储）

3. 知识库管理:
    - 系统会对文献元数据进行向量化处理，构建知识库

4. AI 问答机器人:
    - 基于构建的知识库，提供智能问答服务
