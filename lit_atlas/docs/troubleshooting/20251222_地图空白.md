# 故障复盘报告
- **日期**: 2025-12-22
- **严重级别**: High

## 1. 问题现象
主页地图层完全透明，黑白底图与路径均不可见，仅控制台日志持续输出 Feature 加载信息。

## 2. 根本原因 (Root Cause)
自定义墨迹层在 postrender 中强制使用 globalCompositeOperation = 'multiply'。由于该图层初始像素透明，multiply 与 0 相乘导致所有描绘结果被清零，所以路网与涟漪即便渲染也不可见。

## 3. 解决方案
将混合模式改为默认的 source-over，确保在透明 Canvas 上正常叠画，同时保留其他阴影/透明度效果来模拟墨迹质感。

## 4. 预防措施
- 渲染层变更后必须在浏览器中确认底图与覆盖层可见。
- 如需特殊混合模式，先填充基底或单元测试渲染输出，避免再次出现整层被消除的情况。
